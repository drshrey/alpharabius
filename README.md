# alpharabi.us
Chrome extension for language learning


## Installation
```
npm install
npm install gulp -g
gulp watch
```

## Running + Testing

1. go to [chrome://extensions](chrome://extensions)
2. click on `load unpackaged extension`
3. find build folder that's generated from `gulp watch` and select that
